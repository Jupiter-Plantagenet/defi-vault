Upgradable DeFi Vault

ERC-4626 compliant yield vault with UUPS proxy upgradeability, invariant fuzz testing, and automated static analysis.

Solidity ^0.8.24 OpenZeppelin v5 Foundry Slither GitHub Actions CI

Architecture

User ──deposit──► ERC1967Proxy ──delegatecall──► VaultV1 (ERC-4626)
                      │                              │
                      │                     ┌────────┴────────┐
                      │                     │  MockYieldSource │
                      │                     │  (yield strategy)│
                      │                     └─────────────────┘
                 upgradeTo(VaultV2)
                      │
                      ▼
                  VaultV2 (+ performance fees)

Key Features

ERC-4626 Standard Fully compliant vault interface — deposit, withdraw, redeem, mint with share accounting.
UUPS Upgradeability Seamlessly upgrade vault logic (V1 → V2) without moving user funds. Owner-gated.
Invariant Fuzz Testing 32,768+ random calls verify solvency, exchange rate consistency, and no value leaks.
Slither Static Analysis Automated vulnerability scanning on every PR via GitHub Actions CI/CD.

Test Results

16/16 tests passed — 12 unit tests + 4 invariant fuzz tests (128 runs, 32,768 calls, 0 reverts)
